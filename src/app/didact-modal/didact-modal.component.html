Comment jouer au Tic Tac Toe

Le Tic Tac Toe est un jeu joué sur une grille de 3x3 cases.

Deux joueurs jouent à tour de rôle en plaçant leur symbole (X ou O) dans une case vide.

Le premier joueur à aligner 3 de ses symboles horizontalement, verticalement ou en diagonale gagne la partie.

Si toutes les cases sont remplies et qu'aucun joueur n'a gagné, la partie est déclarée nulle.

Le joueur qui place le symbole X commence la partie.

Pour placer un symbole, cliquez simplement sur la case où vous voulez le placer.

Une fois qu'un symbole est placé, il ne peut pas être déplacé.

Le jeu se poursuit jusqu'à ce qu'un joueur gagne ou que toutes les cases soient remplies.


Amusez-vous bien !

Comment jouer au Puissance 4

Deux joueurs placent à tour de rôle leur jeton de couleur (rouge ou jaune) dans une colonne de leur choix.

Le jeton descend jusqu'à la position la plus basse disponible dans cette colonne.

Le premier joueur à aligner 4 de ses jetons horizontalement, verticalement ou en diagonale gagne la partie.

Si toutes les colonnes sont remplies et qu'aucun joueur n'a gagné, la partie est déclarée nulle.

Pour placer un jeton, cliquez simplement sur la colonne où vous voulez le placer.

Une fois qu'un jeton est placé, il ne peut pas être déplacé.

Le jeu se poursuit jusqu'à ce qu'un joueur gagne ou que toutes les colonnes soient remplies.


Amusez-vous bien !

Comment jouer à Pong

Pong est un jeu de tennis de table simulé en 2D.

Deux joueurs contrôlent chacun une palette qui peut se déplacer horizontalement le long du bord de l'écran.

Le jeu commence avec la balle au centre de l'écran, se déplaçant dans une direction aléatoire.

Les joueurs utilisent leurs palettes pour renvoyer la balle de l'autre côté du terrain.

Si un joueur manque la balle, l'autre joueur marque un point.

Le jeu continue jusqu'à ce qu'un joueur atteigne le nombre de points nécessaire pour gagner.

Utilisez les touches fléchées haut et bas pour déplacer votre palette.

Comment jouer au Démineur

Le Démineur est un jeu de puzzle classique où vous devez révéler toutes les cases sans bombes.

Vous pouvez pour cela utiliser les chiffres des cases découvertes qui indiquent le nombre de bombes sur les cases adjacentes.

Si vous découvrez une case avec une bombe, vous perdez la partie.

Vous pouvez poser des drapeaux sur les cases que vous pensez être des bombes. Cela vous empêchera d'appuyer accidentellement dessus

Le jeu se termine lorsque vous avez découvert toutes les cases sans bombes.

Sur ordinateur, utilisez le clic gauche pour découvrir une case et le clic droit pour poser un drapeau.

Sur téléphone ou tablette, appuyez sur une case pour la découvrir, ou restez appuyé pour poser un drapeau.


Amusez-vous bien !

Règles du jeu du Pendu

Le jeu du Pendu est un jeu de lettres classique qui peut être joué par deux joueurs ou plus. Le but du jeu est de deviner un mot en trouvant les lettres qui le composent. Voici les règles détaillées du jeu :

  1. Un joueur choisit un mot et écrit le nombre de lettres sur une feuille de papier ou un tableau. Les autres joueurs ne doivent pas voir le mot.

  2. Les autres joueurs doivent deviner les lettres qui composent le mot. Pour cela, ils proposent une lettre à tour de rôle.

  3. Si la lettre proposée se trouve dans le mot, le joueur qui a choisi le mot l'écrit à la bonne place sur la feuille ou le tableau.

  4. Si la lettre proposée ne se trouve pas dans le mot, le joueur qui a choisi le mot dessine une partie du corps d'un bonhomme pendu sur la feuille ou le tableau. Le bonhomme pendu est généralement composé de 7 parties : la tête, le corps, les deux bras et les deux jambes.

  5. Les joueurs continuent de proposer des lettres jusqu'à ce que le mot soit deviné ou que le bonhomme pendu soit complet.

  6. Si le mot est deviné avant que le bonhomme pendu ne soit complet, les joueurs qui ont deviné le mot gagnent la partie.

  7. Si le bonhomme pendu est complet avant que le mot soit deviné, le joueur qui a choisi le mot gagne la partie.

Le jeu du Pendu est un excellent moyen de travailler son vocabulaire et sa mémoire tout en s'amusant !

Comment jouer à Flappy Bird

Flappy Bird est un jeu où vous devez aider un oiseau à voler entre des tuyaux sans les toucher.

Pour faire voler l'oiseau, appuyez simplement sur l'écran. Plus vous appuyez longtemps, plus l'oiseau monte haut.

Évitez les tuyaux en faisant passer l'oiseau entre les espaces vides. Si vous touchez un tuyau, la partie est terminée.

Chaque fois que vous passez entre deux tuyaux, vous gagnez un point.

Le jeu devient de plus en plus difficile à mesure que vous avancez, avec des tuyaux plus rapprochés et plus hauts.

Essayez d'obtenir le score le plus élevé possible et de battre vos amis !

Chaque fois que vous surpassez votre score de 3 vous gagnez 10 de points


Amusez-vous bien !
